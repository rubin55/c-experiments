 Practice Problem 2.3
====================


Question
--------

Fill in the blank entries in the following table, giving the decimal, binary
and hexidecimal representations of different byte patterns:

|dec  | bin      | hex      |
|----:|---------:|---------:|
|    0| 0000 0000|       0x0|
|  167|          |          |
|   62|          |          |
|  188|          |          |
|     | 1000 1000|          |
|     | 1111 0011|          |


Answer
------

|dec  | bin      | hex      |
|----:|---------:|---------:|
|    0| 0000 0000|       0x0|
|  167| 1010 0111|      0xA7|
|   62| 0011 1110|      0x3E|
|  188| 1011 1100|      0xBC|
|  136| 1000 1000|      0x88|
|  243| 1111 0011|      0xF3|
