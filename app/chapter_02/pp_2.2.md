 Practice Problem 2.2
====================


Question
--------

Fill in the blank entries in the following table, giving the decimal and hex
representations of different powers of 2:

|n    | 2ⁿ (dec) | 2ⁿ (hex) |
|----:|---------:|---------:|
|    9|       512|     0x200|
|   19|          |          |
|     |          |   0x10000|
|   17|          |          |
|     |        32|          |
|     |          |      0x80|

Answer
------

|n    | 2ⁿ (dec) | 2ⁿ (hex) |
|----:|---------:|---------:|
|    9|       512|     0x200|
|   19|    524288|   0x80000|
|   16|     65536|   0x10000|
|   17|    131072|   0x20000|
|    5|        32|      0x20|
|    7|       128|      0x80|
